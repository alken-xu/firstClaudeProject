## スライド 7｜自律的なトラブル解決

### 例: `better-sqlite3` のインストール失敗

```
エラー: Node.js v22向けのビルド済みバイナリが存在しない
```

### Claudeがやったこと（ユーザーは何もしていない）

```
1. エラーログを読んで原因を特定
   → "prebuild binary not found"
2. npmパッケージのバージョン履歴を調査
   → v9.6.0 ではなく v12.8.0 が対応
3. バージョン指定でインストール
   → npm install better-sqlite3@12.8.0
4. 動作確認 → 解決
```

### ポイント
> **ユーザーは「エラーが出ました」と伝えるだけでよい**  
> エラーログの読み解き・調査・修正・確認まですべて自律実行

## スライド 8｜画像収集・外部サービス連携

### 「雰囲気の近い画像を見つかれば良いです」の一言から

Claudeが連続して実行したこと:

```
1. WebSearch で Unsplash の Photo ID を検索
   （テーマ別に並行検索: 客室・温泉・料理・季節）
2. ダウンロードURLを一覧で提示
   → ユーザーが内容を確認（セキュリティチェック）
3. 20枚を並行ダウンロード → public/images/ に配置
4. フロントエンドのプレースホルダーを <img> タグに差し替え
5. ヒーロー部分を5秒自動切替スライダーに改修
```

### ポイント
- 検索 → ダウンロード → コード修正 → 確認 が**一気通貫**
- 「内容を確認してからダウンロード」という**ユーザーの安全確認に対応**
- Unsplashの直接fetchがエラー（403）→ WebSearchで代替 と**自ら判断**
